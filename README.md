# Exercicios_python
Resolução de questões da OBMEP (Olimpiada Brasileira de Matemática das Escolas Públicas) exercitando a lógica de programação e a própria linguagem.

## Metodologia:
primeiro o código com ex00_.py é o problema sendo resolvido por IA, dessa forma obtendo uma direção a ser seguida para assim criar meu próprio código sendo o notes00_.py

## Disclaimer:
O código feito por IA é _APENAS_ para fins de comparação, em nenhum momento foi copiado qualquer linha de código ou usado a lógica do código criado por inteligência artificial. Logo após a criação do código, foi submetido a avaliação do Gemini e assim futuramente serão feitas atualizações a medida que ocorrer a evolução do programador.

### Obs:
Bibliotecas utilizadas:
| Biblioteca   | Para quê usar                                                  |
| ------------ | -------------------------------------------------------------- |
| `math`       | raiz, mínimo, máximo, trigonometria                            |
| `itertools`  | permutações, combinações, produto cartesiano (ouro para OBMEP) |
| `functools`  | `lru_cache` para recursão                                      |
| `statistics` | média, moda, variância                                         |
| `numpy`      | matrizes grandes                                               |
| `matplotlib` | visualizar padrões                                             |

# Exercicios_python
Resolução de questões da OBMEP (Olimpiada Brasileira de Matemática das Escolas Públicas) exercitando a lógica de programação e a própria linguagem.

### Obs:
Bibliotecas utilizadas:
| Biblioteca   | Para quê usar                                                  |
| ------------ | -------------------------------------------------------------- |
| `math`       | raiz, mínimo, máximo, trigonometria                            |
| `itertools`  | permutações, combinações, produto cartesiano (ouro para OBMEP) |
| `functools`  | `lru_cache` para recursão                                      |
| `statistics` | média, moda, variância                                         |
| `numpy`      | matrizes grandes                                               |
| `matplotlib` | visualizar padrões                                             |
